# task_03

## How to use
```
sudo apt install freeglut3-dev  
sudo apt install libglew-dev  
sudo apt install libglu1-mesa-dev  
git clone https://github.com/grihabor/task_03
cd task_03
```
To build the project:
```
sudo make all
```
To run the project:
```
make run
```
To clean the project:
```
make clean
```
